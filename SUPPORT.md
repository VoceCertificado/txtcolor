# Support

Thank you for using TXT Color.

TXT Color is a simple syntax highlighting extension for plain text, logs, configuration, environment and properties files.

## How to get support

If you find a problem, have a suggestion, or want to request support for another file type, please open an issue on GitHub:

https://github.com/VoceCertificado/txtcolor/issues

## Before opening an issue

Please include:

- Extension version
- Editor name and version
- Operating system
- File extension
- A small text example
- Screenshot, if possible

## Feature requests

Feature requests are welcome, especially for:

- Additional file extensions
- LOG file highlighting
- ENV file highlighting
- INI, CONF, CFG and PROPERTIES improvements
